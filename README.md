# Guido Magi - Portfolio

Portfolio personal desarrollado con React, desplegado en GitHub Pages.

[![Live Demo](https://img.shields.io/badge/demo-live-green?style=flat-square)](https://guidomagi23.github.io/miportfolio)

## Preview

> [guidomagi23.github.io/miportfolio](https://guidomagi23.github.io/miportfolio)

## Tech Stack

- **React 18** — UI library
- **React Router v6** — Navegación con HashRouter
- **React Carousel** — Sliders interactivos
- **GitHub Pages** — Hosting y deploy

## Estructura

```
src/
├── components/
│   ├── cover/        # Hero / portada
│   ├── navbar/       # Barra de navegación
│   ├── about/        # Sección sobre mí
│   ├── slider/       # Sliders de proyectos
│   ├── info/         # Sección de información
│   └── footer/       # Pie de página
├── App.js
└── index.js
```

## Instalación

```bash
git clone https://github.com/guidomagi23/miportfolio.git
cd miportfolio
npm install
```

## Desarrollo

```bash
npm start
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.

## Deploy a GitHub Pages

```bash
npm run deploy
```

Esto ejecuta `build` automáticamente y publica en la rama `gh-pages`.
